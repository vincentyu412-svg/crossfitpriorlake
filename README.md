# CrossFit Prior Lake Website

A modern, responsive website for CrossFit Prior Lake gym.

## Pages

- **Home** (`index.html`) – Hero section, features, about preview, programs, testimonials, CTA
- **About** (`about.html`) – Gym story, mission/values, coach bios
- **Membership** (`membership.html`) – Pricing plans, add-ons, FAQ
- **Contact** (`contact.html`) – Contact form, info, and map placeholder

## Features

- Fully responsive design (mobile, tablet, desktop)
- Dark theme with red accent colors
- Smooth scroll animations
- Mobile hamburger menu
- Scroll-triggered counter animations
- Contact form with validation
- No frameworks – pure HTML, CSS, JavaScript

## Getting Started

1. Clone this repository
2. Add your own images to the `images/` folder
3. Replace placeholder content (phone, address, coach names, etc.) with real info
4. Replace the map placeholder in `contact.html` with a Google Maps embed
5. Deploy via GitHub Pages, Netlify, or any static hosting

## Customization

- **Colors**: Edit CSS variables at the top of `css/style.css`
- **Fonts**: Loaded from Google Fonts (Oswald + Open Sans)
- **Icons**: Font Awesome 6 (loaded via CDN)

## Deployment (GitHub Pages)

1. Push to GitHub
2. Go to Settings → Pages
3. Set source to `main` branch, root folder
4. Your site will be live at `https://yourusername.github.io/crossfitpriorlake/`
